# Employee Attendance System

Simple employee attendance web application built with Node.js and SQLite.

## Features

- Employee Check-in
- Attendance Dashboard
- Attendance Table
- Simple Statistics

## Tech Stack

Node.js  
Express.js  
SQLite  
HTML CSS JavaScript  

## Installation

```bash
npm install
node server.js
```

Open in browser:

```
http://localhost:3000
```
